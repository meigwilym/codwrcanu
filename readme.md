# Y Codwr Canu

App HTML5 i godi'r canu. Ar gyfer unrhyw sefyllfa ble mae cydganu. 

Dyma'r côd ar gyer yr app a sut mae o'n gweithio. 

## Esboniad

Index.php sy'n gwneud y gwaith. Mae'r caneuon yn caneuon.json fel un object json. canu.appcache i wneud y cachio. sql.php i dynnu'r data o'r bas data lleol.
